# School Indoor Navigation System

A web-based indoor navigation system designed to help students, staff, and visitors easily find rooms, offices, and facilities within a school campus using interactive indoor maps.

This project focuses on **frontend-only implementation**, inspired by platforms such as **MazeMap**, and uses embedded indoor maps created with **Mappedin**.

---

## üìå Project Overview

Navigating large school buildings can be difficult, especially for new students and visitors.  
This system provides a simple and intuitive web interface where users can:

- View an interactive indoor map of the school
- Search for rooms and locations
- Switch between **2D and 3D map views**
- Navigate floors visually using a modern dark-themed UI

The application runs entirely in the browser and **does not require a backend server or database**.
