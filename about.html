About This Project

A smart campus navigation solution for ICT University

Project Overview

This web-based navigation system is designed to help users locate buildings, classrooms, and important services within ICT University.

The system is built using HTML, CSS, SVG, Mappdin and JavaScript with a focus on simplicity, performance, and usability.

Our Approach

Hello
